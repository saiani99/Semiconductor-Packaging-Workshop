# Semiconductor-Packaging-Workshop
This repository is a summary of the semiconductor packaging workshop offered by VSD. It includes details on the lectures and labs presented as part of the workshop.
Module 1: Packaging evolution: From basics to 3D integration

L0: Introduction to semiconductor packaging and industry overview

This lecture explains why semiconductor packaging is needed and how the packaging and testing industry functions. It also discusses some of the companies involved in this process.

Packaging is needed to protect chips which are made in a protected environment to function in the real world. 
Packaging is needed to allow the dies to work in real world conditions by protecting it from corrosion, moisture and physical damage. 
The personality of the die is brought by the packaging.
